# Hello

## Run locally

**Main site** (portfolio + Clean layout): open [`index.html`](index.html) in a browser, or from the repo root:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000/`. Some features behave best over HTTP (not `file://`).

Use **Layout → Clean** in the header (or **Switch to clean view** in the hero) for the minimal layout.
